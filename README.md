# 3D-Gradient-Engine

3D engine with moving gradient

Used:

SDK: DirectX 11 
IDE: Visual studio 2022 
API for window: Windows API 
System: Windows 10 

Controls:
Movement: W/S/A/D/SPACE/L-CTRL 
Look Left/Right: Left Arrow, Right Arrow
